¿Motor paso a paso (Stepper) o Servomotor?

MM
MicroManiac Publicado hace 7 horas

Para un proyecto que necesita un movimiento angular preciso, no sé si usar un motor paso a paso (como el 28BYJ-48) o un servomotor (como el SG90). ¿Cuál es la diferencia fundamental y cuándo conviene usar cada uno?

Respuestas

RS
RoboSpark Publicado hace 4 horas

Simple: un servomotor es para ir a una posición y mantenerla (ej. de 0 a 180 grados). Tiene un sistema interno que sabe en qué ángulo está. Un motor paso a paso es para girar una cantidad precisa de pasos, pero no sabe su posición absoluta a menos que tengas un sensor externo. El servo es más fácil para ángulos fijos (un brazo robótico), el stepper es mejor para rotación continua y precisa (una impresora 3D).
