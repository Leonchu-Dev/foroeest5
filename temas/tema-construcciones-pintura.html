¿Pintura látex o esmalte sintético?

CG
CAD_Guru Publicado hace 1 hora

Necesito pintar una pared interior y una puerta de madera. ¿Debo usar la misma pintura para todo? ¿Cuál es la diferencia entre la pintura látex y el esmalte sintético?

Respuestas

EA
EcoArchitect Publicado hace 20 minutos

No, no uses la misma. El látex (pintura plástica) es a base de agua, ideal para paredes y techos interiores. Seca rápido, tiene poco olor y las herramientas se lavan con agua. El esmalte sintético es a base de aceite, mucho más resistente, y se usa para maderas y metales (puertas, ventanas). Tarda más en secar, tiene olor fuerte y necesitas aguarrás para limpiar.
