¿Qué es exactamente una API y cómo se usa?

PD
PixelDreamer Publicado hace 15 horas

Escucho por todos lados la palabra API (API de Google Maps, API de Twitter, etc.) pero no termino de entender qué es exactamente. ¿Es como una librería? ¿Cómo se usa en un proyecto web?

Respuestas

SS
ServerSage Publicado hace 10 horas

Buena pregunta. Imagina que una API es como el mozo de un restaurante. Tú (tu aplicación) le pides algo del menú (los datos o funciones que ofrece la API), el mozo va a la cocina (el servidor), trae tu pedido y te lo entrega. No necesitas saber cómo se cocina, solo cómo pedir. En la web, se usa haciendo "peticiones HTTP" a una URL específica y la API te devuelve los datos, usualmente en formato JSON.
