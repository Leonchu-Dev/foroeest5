¿Cómo funciona el PWM para controlar un LED o un motor?

PP
PowerPulse Publicado hace 3 horas

Hola, sé que con `analogWrite()` en Arduino se genera una señal PWM, y que sirve para variar el brillo de un LED. Pero no entiendo cómo funciona. ¿El pin entrega un voltaje variable o es otra cosa?

Respuestas

PJ
Prof_Juarez Publicado hace 1 hora

No entrega un voltaje variable. Un pin digital solo puede estar en 0V o 5V. Lo que hace el PWM (Modulación por Ancho de Pulso) es prender y apagar el pin muy rápidamente. El 'duty cycle' es el porcentaje de tiempo que el pin está en 5V. Si está el 50% del tiempo prendido, el LED brilla a la mitad. Si está el 100%, brilla al máximo. El ojo humano (o la inercia de un motor) promedia este "parpadeo" y lo percibe como un nivel de intensidad continuo.
